# Allurite CRM

Allurite CRM is a state-of-the-art enterprise-grade Customer Relationship Management and Account Orchestration application optimized for the Egyptian market environment.

---

## 🌟 Key Features

1. **Dashboard & KPIs**: Real-time business telemetry cards, chronologically planned followups, assigned workforce priorities, and productivity leaderboard panels.
2. **Staff Directory**: Complete employee profiles management, customizable departments directory, and job roles permissions matrix.
3. **Client Pipeline**: Pipelines tracking lead conversions, logs interaction notes, and assigned agents management.
4. **Follow-Up Planner**: Calendar planners for touchpoint meetups, automatic reminders, and outcome notes log entries.
5. **Workforce Tasks**: Collaborative Kanban board supporting priorities, comments, deliverables attachments, review requests, and approvals.
6. **Notification Center**: Real-time notifications and alerts gating configurations.
7. **Cloud Files Manager**: Secure Vercel Blob cloud storage files directory supporting categories, tags, and archives.
8. **Reports & Analytics**: Custom operational metrics exporters generating CSV spreadsheets.
9. **System Backup & Recovery**: One-click database snapshot creators and recovery operations for SuperAdmins.
10. **Security & Audit Logs**: High-security session tracking logs, lockouts, and chronological event timelines.
11. **Localization & RTL**: Full Arabic-first interface with dynamic English switcher and automated RTL layout orientations.

---

## 🛠️ Technology Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router & React Server Components)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Database**: [MongoDB](https://www.mongodb.com/) (ODM: Mongoose)
- **Styling**: Modern CSS custom properties (variables)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Storage**: [Vercel Blob Storage](https://vercel.com/docs/storage/vercel-blob)
- **Deployment**: [Vercel](https://vercel.com/)
